Non posso ancora migliorare davvero il README “ready-to-paste”, perché con lo strumento **Repository** disponibile nel tuo ambiente riesco solo a ottenere *metadati* del repository (titolo, descrizione, branch, ecc.), ma **non posso leggere i file** (`README.md` e `package.json`) né tantomeno la sezione `scripts`.

Per procedere, incollami qui (come hai già chiesto, ma serve proprio il contenuto reale):

1) **tutto il contenuto attuale di `README.md`**  
2) **solo** la sezione `scripts` di `package.json`

Appena li mandi, te lo riscrivo in **un unico blocco markdown** coerente con i comandi reali definiti in `package.json` (install, usage, build/test, ecc.).
