Non riesco a soddisfare la richiesta così com’è: lo strumento `Repository` che posso usare qui mi restituisce **solo metadati** della repo (nome, branch, license, ecc.) e **non** il contenuto dei file, quindi **non posso leggere**:

1) il contenuto attuale di `README.md`  
2) la sezione `"scripts": { ... }` di `package.json`

Perciò non posso “incollare esattamente” quei contenuti come richiesto.

Se mi dai tu:
- il contenuto attuale di `README.md` (incolla qui tutto), **e**
- la sezione `"scripts": { ... }` di `package.json` (incolla solo quella),

allora ti preparo immediatamente:
- un **README.md completo migliorato** pronto da copiare/incollare
- una sezione **Build/Run** perfettamente coerente con gli `scripts`.

In alternativa, se il tuo ambiente offre un tool diverso (es. “read file” / “contents_url” / download di file) fammelo sapere: con quello potrei farlo da solo.
